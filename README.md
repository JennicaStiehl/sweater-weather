# Sweater Weather
## Description
```Consumes Google Geocoding API and DarkSky API
```
## Getting Started/Requirements/Prerequisites/Dependencies
```
Ruby Version: 2.4.1
run the following commands:

bundle install
bundle exec figaro install
bundle exec rake db:{create,migrate,seed}
```
